Wordpress 3 brought many great features. One of those great features is **navigation menu** support. Although many themes not supporting this great feature yet, you can use it with **dropdown menu widget** to create **custom menus** on your site. In this tutorial I will explain you how to use custom navigation menus with dropdown menu widget.
 ***

***

First make sure you have latest wordpress installation. Custom Navigation Menu support comes with Wordpress 3. So if your wordpress version is lower than that, you need to update first. Don't worry, it will worth it ;)

If you have the latest wordpress installation now, you will see **Menus** link under **Appearance** section of the admin panel.

<a href="{{ site.imgbase }}/custom-menu-dropdown-01.jpg"><img src="{{ site.imgbase }}/custom-menu-dropdown-01.jpg" alt="" title="custom-menu-dropdown-01" width="560" height="300" class="aligncenter size-full wp-image-1034" /></a>

Click on **Menus** to start building our brand new custom menu. Wordpress may warn you about "theme doesn't support custom menus" but don't worry. We will be using menu in our dropdown widget so it doesn't matter.

<a href="{{ site.imgbase }}/custom-menu-dropdown-02.jpg"><img src="{{ site.imgbase }}/custom-menu-dropdown-02.jpg" alt="" title="custom-menu-dropdown-01" width="560" height="300" class="aligncenter size-full wp-image-1034" /></a>

You see white box there under + sign. Enter a name for our new menu. For example, let's call it, "My Custom Dropdown". How original! Click on create menu button afterwards to create the menu.

<a href="{{ site.imgbase }}/custom-menu-dropdown-03.jpg"><img src="{{ site.imgbase }}/custom-menu-dropdown-03.jpg" alt="" title="custom-menu-dropdown-01" width="560" height="300" class="aligncenter size-full wp-image-1034" /></a>

Your menu is created now but it doesn't have any link inside. You will see tools on the left to add links to your custom menu. You can add blog pages & categories to your menu.

<a href="{{ site.imgbase }}/custom-menu-dropdown-03_1.jpg"><img src="{{ site.imgbase }}/custom-menu-dropdown-03_1.jpg" alt="" title="custom-menu-dropdown-03_1" width="560" height="300" class="aligncenter size-full wp-image-1043" /></a>

You can also add custom links to your menu. After you add links, you can easily order your menu items using **drag and drop**.

<a href="{{ site.imgbase }}/custom-menu-dropdown-08.jpg"><img src="{{ site.imgbase }}/custom-menu-dropdown-08.jpg" alt="" title="custom-menu-dropdown-01" width="560" height="300" class="aligncenter size-full wp-image-1034" /></a>

You can create sub menus dragging a link item under another.

<a href="{{ site.imgbase }}/custom-menu-dropdown-04.jpg"><img src="{{ site.imgbase }}/custom-menu-dropdown-04.jpg" alt="" title="custom-menu-dropdown-01" width="560" height="300" class="aligncenter size-full wp-image-1034" /></a>

 After your ordering is complete, save your menu. This menu is now added to dropdown menu widget. Steps from now on requires <a href="{{ site.baseurl }}/projects/wordpress/dropdown-menu-widget/">dropdown menu widget</a> installed on your site. If you haven't already <a href="{{ site.baseurl }}/projects/wordpress/dropdown-menu-widget/">click here to get it</a>.

<a href="{{ site.imgbase }}/custom-menu-dropdown-05.jpg"><img src="{{ site.imgbase }}/custom-menu-dropdown-05.jpg" alt="" title="custom-menu-dropdown-01" width="560" height="300" class="aligncenter size-full wp-image-1034" /></a>

Add a dropdown menu widget to your widget area. I like <a href="{{ site.baseurl }}/wordpress/themes/">my custom themes</a> to have it. Makes wordpress more customizable.

<div class="note">Note that not all the themes have header widget areas. You can either use <a href="{{ site.baseurl }}/2773/dropdown-menu-widget-template-tag-usage-explained/" title="Dropdown Menu Widget template tag usage" target="_blank">Dropdown Menu PHP template tag</a> or you can <a href="{{ site.baseurl }}/727/how-to-widgetize-your-theme-in-3-simple-steps/" title="How to add a header widget area to your wordpress theme" target="_blank">add a header widget area</a> to your theme to overcome this. I recommend using **PHP Template tag** instead.</div>

<a href="{{ site.imgbase }}/custom-menu-dropdown-06.jpg"><img src="{{ site.imgbase }}/custom-menu-dropdown-06.jpg" alt="" title="custom-menu-dropdown-01" width="560" height="300" class="aligncenter size-full wp-image-1034" /></a>

Select your custom menu from the Menu selection of dropdown menu widget. And click on *Save*.

<a href="{{ site.imgbase }}/custom-menu-dropdown-07.jpg"><img src="{{ site.imgbase }}/custom-menu-dropdown-07.jpg" alt="" title="custom-menu-dropdown-01" width="560" height="300" class="aligncenter size-full wp-image-1034" /></a>

If you just installed the dropdown menu widget you may need to visit menu style page. Change dropdown menu theme to any theme you like and then click on Save Changes.

<a href="{{ site.imgbase }}/custom-menu-dropdown-09.jpg"><img src="{{ site.imgbase }}/custom-menu-dropdown-09.jpg" alt="" title="custom-menu-dropdown-01" width="560" height="300" class="aligncenter size-full wp-image-1034" /></a>

Now visit your blog and enjoy your new custom menu. Cheers!
